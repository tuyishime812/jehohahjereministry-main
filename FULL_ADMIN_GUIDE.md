# Jehovah Jire Ministry - Full Admin Guide

This guide explains how to use the complete admin system for managing your Jehovah Jire Ministry website.

## Getting Started

1. Open your browser and navigate to `admin.html`
2. Login with:
   - Username: `jehovah`
   - Password: `jehovah123`
3. You'll be taken to the professional CMS dashboard

## Managing Page Content

### Editing Page Content
1. Click on the tab for the page you want to edit (Home, About, Contact, etc.)
2. Use the rich text editor to make changes to the content
3. Use the toolbar buttons to format text:
   - B (Bold) - Make text bold
   - I (Italic) - Make text italic
   - Heading buttons - Create headings (H1, H2, H3)
   - List buttons - Create bullet or numbered lists
   - Link button - Add hyperlinks
   - Image button - Insert images from your computer
4. When finished, click "Save Changes"

### Adding Images
1. Click the image icon in the editor toolbar
2. Select an image from your computer/device
3. The image will be inserted at your cursor position
4. Click "Save Changes" to store the content with the image

## Managing Contact Information

1. Click on the "Contact Info" tab
2. Add or edit phone numbers in the phone number fields
3. Add or edit WhatsApp numbers in the WhatsApp fields
4. Click "Save Contact Info" to store the changes

## Managing Team Members

1. Click on the "Team Members" tab
2. Add new team members using the "Add Member" button
3. Edit existing team members by modifying the fields
4. Click "Save Team Members" to store the changes

## Managing Testimonials

1. Click on the "Testimonials" tab
2. Add new testimonials using the "Add Testimonial" button
3. Edit existing testimonials by modifying the fields
4. Click "Save Testimonials" to store the changes

## Managing Site Settings

1. Click on the "Site Settings" tab
2. Update the site title and description
3. Click "Save Settings" to store the changes

## Managing Media

1. Click on the "Media Management" tab
2. Upload a new logo using the logo upload section
3. Upload a new hero background using the background upload section
4. Changes are saved automatically when you upload images

## Publishing Changes

Changes are automatically saved to your Supabase database and will appear on your website immediately. You can verify changes by:

1. Clicking "Refresh Preview" in the admin panel
2. Visiting your website in a new tab
3. The changes will be visible immediately

## Security Best Practices

1. Keep your admin credentials secure
2. Regenerate your Supabase API keys periodically
3. Don't share your admin login details
4. Log out when finished using the admin panel

## Troubleshooting

- If content doesn't appear on the website, try refreshing the page
- If images don't show, make sure they were uploaded successfully
- If you get connection errors, check your Supabase configuration
- Clear your browser cache if changes don't appear

## Support

If you encounter issues with the system, please contact your web developer or refer to the Supabase documentation for database-related issues.
