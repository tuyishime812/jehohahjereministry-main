# JehoHahJere Ministry - Getting Started Guide

Welcome to your new ministry website with admin capabilities! This guide will help you get started with managing your content.

## Quick Start

### 1. Access the Admin Panel
- Double-click the `admin-panel.bat` file OR
- Open your browser and navigate to `admin.html`
- Enter password: `admin123`
- Click "Login" to access the professional CMS dashboard

### 2. Edit Content
- On the admin dashboard, you'll see text areas for each page
- Type or paste your content into the appropriate section
- Click "Save" to store your changes
- Changes are saved automatically to your browser

### 3. Upload Images
- Use the image upload sections to add:
  - Custom logo
  - Custom hero background
- Select an image file and click "Upload"
- The image will be applied to your website immediately

### 4. Preview Changes
- Use the "Preview" section to see how your content will look
- Or click "View [Page Name] Page" to see the live page

## Important Notes

- All content is stored in your browser's localStorage
- If you clear your browser data, you'll lose all custom content
- Content will appear differently in different browsers
- For best results, use Chrome, Firefox, or Edge

## Troubleshooting

Q: My changes aren't showing up on the website
A: Refresh the page (Ctrl+F5) to see the latest changes

Q: I forgot the admin password
A: The default password is "admin123"

Q: I cleared my browser data and lost my content
A: Unfortunately, content stored in browser localStorage is lost when browser data is cleared

## Need Help?

If you need assistance with your website:
1. Check that all HTML files have the content manager script
2. Ensure your browser supports localStorage
3. Contact your website developer for advanced issues
